# Fumiya Morishita Portfolio (Static)

Appleライクな余白・タイポグラフィを意識した、1ページ構成の静的ポートフォリオです。

## 構成ファイル

- `index.html`
- `styles.css`
- `main.js`
- `public/favicon.svg`
- `public/og-image.svg`
- `public/robots.txt`

## ローカル起動

```bash
python3 -m http.server 4173
```

ブラウザで `http://localhost:4173/index.html` を開いて確認してください。
